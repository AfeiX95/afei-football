# 视频文案矩阵：图文→视频转换系统

> **一次选题，两种输出。** 图文和视频共享选题、共享数据源，但输出形式不同——文章堆信息密度，视频走线性叙事。

---

## 一、核心设计

### 1.1 视频类型一览

| 类型 | 时长 | 频率 | 内容来源 | 制作成本 |
|------|------|------|---------|---------|
| **赛前前瞻口播** | 2-3分钟 | 每篇文配1条 | 当天前瞻文章 | 低 |
| **热点快评口播** | 1-2分钟 | 随热点 | 微头条/争议话题 | 低 |

**核心定位：** 纯口播，不做画面匹配。录好音对口播稿，配一张封面图就能发。

### 1.2 文章→视频转换原则（3条铁律）

| # | 原则 | 说明 |
|---|------|------|
| 1 | **保留钩子，砍掉推导** | 文章里的数据推导过程全部砍掉，只留结论和最有冲击力的1-2个数字 |
| 2 | **一句话一个信息** | 文章能写长句嵌套，口播必须一句一个信息点。每句话不超过20字 |
| 3 | **外媒→分歧对撞** | 外媒预测从罗列改为只留分歧最大的2个，做成"卫报说X，队报说Y"的对撞结构 |

## 二、风格→口播结构映射

你的7种文章风格，每种对应一种口播叙事结构。所有结构按2-3分钟安排。

### 2.1 数据先导式

**适用场景**：文章有一个爆炸性反差数据

**口播叙事线**：
```
① 抛数字开场（1句话） → ② 用2个对比数据展开 → ③ 主队情况（3-4句）
→ ④ 客队情况（3-4句） → ⑤ 关键变量 → ⑥ 外媒分歧+数据判断
→ ⑦ 个人倾向 → ⑧ 互动收尾
```

### 2.2 对决体

**适用场景**：风格对撞的比赛（矛vs盾、主场龙vs客场虫）

**口播叙事线**：
```
① 对决概念开场 → ② A方分析（核心优势+隐忧）
→ ③ B方分析（核心优势+隐忧） → ④ 矛盾对撞点
→ ⑤ 数据判断 → ⑥ 个人倾向 → ⑦ 互动收尾
```

### 2.3 分板块式

**适用场景**：信息量大、规整

**口播叙事线**：
```
① 总起（"这场3个看点"） → ② 看点1 → ③ 看点2
→ ④ 看点3 → ⑤ 个人倾向 → ⑥ 互动收尾
```

### 2.4 叙事流

**适用场景**：有赛季故事线/历史渊源的比赛

**口播叙事线**：
```
① 悬念开场 → ② 故事起点（从赛季讲起）
→ ③ 起承转合（数字嵌入叙事） → ④ 当前局势
→ ⑤ 倾向+互动收尾
```

### 2.5 清单体

**适用场景**：中下游混战、快速出稿

**口播叙事线**：
```
① 总起（"为什么值得看？N个理由"）
→ ② 理由1 → ③ 理由2 → ④ 理由3
→ ⑤ 总结+互动收尾
```

### 2.6 战术拆解式

**适用场景**：有明显战术博弈的比赛

**口播叙事线**：
```
① 战术钩子开场 → ② A方怎么打 → ③ B方怎么破
→ ④ 关键对位 → ⑤ 预判谁会先调整 → ⑥ 互动收尾
```

### 2.7 微头条 → 观点快评

**适用场景**：热点话题、争议观点

**口播叙事线**：
```
① 观点直给 → ② 1个数据支撑 → ③ 个人判断 → ④ 站队提问
```

## 三、写作池模块→视频提取规则

你的写作池有7大模块，视频提取规则如下：

| 模块 | 文章版 | 视频版 | 提取比例 |
|------|--------|--------|---------|
| 开头钩子 | 6种钩子选1，200-300字 | 只留最有冲击力的1句，≤20字 | 10-15% |
| 数据板块 | 三层递进/对比式/单点爆破 | 只留最爆炸的1-2个数字，砍掉推导过程 | 5-10% |
| 背景叙事 | 赛季叙事线/搅局者/首回合呼应 | 压缩到1-2句，只留最出人意料的部分 | 10-15% |
| 球星板块 | 故事型/数据型/对位型 | 保留，改口语描述，≤3句 | 15-20% |
| 伤病板块 | 清单式/影响评估式 | 只提影响最大的1个伤停 | 5% |
| 预测板块 | 概率+比分+理由 | 直接给结论（"个人倾向X比X"），不要概率 | 10% |
| 结尾板块 | 4种可选 | 只剩互动提问，不要合规声明 | 5% |

### 提取示例

**文章原文（数据板块-三层递进）：**
> 曼城最近6场比赛5胜1平，进了17个球只丢5个。主场近3场全胜，3场下来只让对手射正4次。英超主场过去40场赢了30场，输？只输了3次。这不是状态好，这是结构性的主场优势——伊蒂哈德就是英超最难啃的骨头。

**视频提取（保留核心+砍推导）：**
> （15s内）"曼城主场过去40场赢了30场。什么叫主场龙？这就是。"

## 四、增强版格式 → 视频版本

你的赛前前瞻有增强版格式（外媒预测+大小球+角球），视频版转换规则：

### 4.1 外媒预测 → 「分歧对撞」

**文章版**：2-5个外媒观点罗列
**视频版**：只留分歧最大的2个

```
❌ 文章写法：
"卫报看好曼城赢球，队报认为平局可能性更大，电讯报支持曼城不败"

✅ 视频写法：
（10s）"外媒怎么说？卫报说曼城稳赢，队报说——这可能是冷门。"
```

**规则**：不能超过2个，必须有分歧（没分歧就不提外媒）。

### 4.2 大小球数据 → 「金句带过」

**文章版**：机构初盘+水位变化+大球率趋势
**视频版**：一句判断

```
"机构开了3.5球的盘，按照两队近期的进球效率……可能不止这个数。"
```

**规则**：只保留1句，放在核心判断前。

### 4.3 角球数据 → 「一句带过」

**文章版**：两队场均角球+近期走势+综合预期
**视频版**：一句话收束，不展开

```
"角球方面，拜仁主场7.8个压着打，科隆这个定位球防守……有隐患。"
```

## 五、口播稿输出格式

每篇视频文案附在文章末尾，格式固定：

### 5.1 标准输出模板

```
## 📹 视频文案（口播稿 · 约X分XX秒）

**视频标题（30字内）：**
[标题]

**视频简介（400字内）：**
[简介]

**口播稿（约X分XX秒）：**

[口播全文，自然分段，一句一行]
```

### 5.2 口播稿写作要点

- **不分秒**：不用标注0-5s、5-18s这类时间码
- **自然分段**：按逻辑段落空行，一段大概讲20-30秒
- **一句话一行**：方便录音时看稿，不会读串行
- **2-3分钟量**：大约350-550字（中文口播速度约180字/分钟）

### 5.3 示例开头（数据先导式）

```
拜仁这赛季进了117个球。比德甲历史纪录多了16个。离谱不离谱？

凯恩一个人33球，比科隆全队的三分之二还多。
117球什么水平？每场3.5个，踢了34轮。
```

## 六、口播降AI——说得像人话

### 6.1 文章→口播的4个关键转换

| 文章写法（阅读感） | 口播写法（说话感） |
|-------------------|------------------|
| "数据显示……" | "数据摆在这：……" |
| "值得注意的是……" | "有意思的是……" |
| "从赛季整体来看……" | "整个赛季看下来……" |
| "综合判断，本场倾向于……" | "要我选，我偏向……" |

### 6.2 口播节奏控制

- 每句话不超过20字（超过就断句）
- 关键数字前停顿0.3秒（"过去40场……赢了30场"）
- 用重复制造强调（"主场龙——真正的主场龙"）

### 6.3 口播检查清单

- [ ] 读一遍出声，顺不顺？
- [ ] 有没有多余的铺垫？（"今天我们来聊一下"→直接删）
- [ ] 数字顺耳吗？（"场均2.3个"→"每场2到3个"）
- [ ] 每句话像"说"不像是"读"

## 七、工作流集成

### 7.1 在11步流程中的位置

视频文案在**写稿完成之后，配图关键词之前**插入：

```
Step 6 → 写稿完成
     ↓
Step 7 → 视频口播稿生成（新增）
     · 从文章提取2-3分钟口播量
     · 按风格映射选叙事线
     · 写标题+简介+口播稿
     · 附在文章末尾
     ↓
Step 8 → 配图关键词（文章配图用，视频不需要）
```

### 7.2 每次产出物

1. **视频标题**（30字内）
2. **视频简介**（400字内）
3. **口播稿**（2-3分钟纯文案）
4. **文章配图关键词**（6张，中英文双语）

视频不需要配图关键词，也不需要画面提示词。

## 八、矩阵速查

### 8.1 文章风格→口播结构速查

| 文章风格 | 口播叙事线 |
|---------|-----------|
| 数据先导式 | ①抛数字②对比展开③主队④客队⑤变量⑥外媒+判断⑦倾向⑧互动 |
| 对决体 | ①对决开场②A方③B方④矛盾点⑤判断⑥倾向⑦互动 |
| 分板块式 | ①总起②看点1③看点2④看点3⑤倾向⑥互动 |
| 叙事流 | ①悬念②故事③数据嵌入④局势⑤倾向+互动 |
| 清单体 | ①总起②理由1-3⑤总结⑥互动 |
| 战术拆解式 | ①战术钩子②A方③B方④对位⑤预判⑥互动 |
| 微头条 | ①观点②数据③判断④互动 |

### 8.2 比赛类型→口播策略

| 比赛类型 | 策略 |
|---------|------|
| 强强对话（争冠/前四） | 叙事流或战术拆解式，2-3分钟 |
| 强弱分明 | 数据先导式，2分钟 |
| 小联赛冷门预警 | 对决体，2分钟 |
| 中下游混战 | 清单体或分板块式，2分钟 |
| 热点/争议话题 | 微头条快评，1-2分钟 |
